## 344. 反转字符串

### 详情

[Leetcode 题目链接](https://leetcode.cn/problems/reverse-string/description/)

### 解法

#### 双指针

- 思路：

- 代码：

  ```js
  const reverseString = (s) => {
      let left = 0, right = s.length - 1
      while (left < right) {
          let sup = s[left]
          s[left] = s[right]
          s[right] = sup
          left ++
          right --
      }
      return s
  }
  ```

- 复杂度

  - 时间复杂度：O(n)
  - 空间复杂度：O(1)
